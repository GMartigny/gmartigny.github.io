You can put whatever you want in it. Image, text or even video can be throw in.
If you want to make it vertical : class="carrousel vertical" and baaam ... vertical slider !

Code for this one : 

<div class="carrousel vertical"><br />
    <img src="http://picsum.photos/100/300" alt=""/>
    <img src="http://picsum.photos/200/200" alt=""/>
    <img src="http://picsum.photos/100/400" alt=""/>
</div>

And for this beauty, you only need a stylesheet and a script file.
Call the script right before your body end tag </body>.
Code you can find on GitHub.
