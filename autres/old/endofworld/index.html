Time to the end of the world :
0%
Explications

La fonction Time() ou Date() est largement utilisée dans les scripts dynamiques et les pages web.
Elle permet de récupérer le nombre de secondes passées depuis "l'ère UNIX" soit le 01/01/1970 à 00:00
Bien que pratique, cette méthode a une grosse faille : lorsque le nombre de seconde dépassera
l'entier maximum qui peut être stocké dans une seule variable de 32 bits (soit 2^31-1 = 2147483647),
qui sait comment l'ensemble des programmes va réagir.
Pour plus de détails : Wikipédia - heure UNIX
Comme pour le bug annoncé de l'an 2000, espérons que ça se passera sans trop de problèmes.
